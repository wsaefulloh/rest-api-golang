## Feature
- CRUD Product
- CRUD Category
- CRUD History
- MVC Models
- Search Product Name
- Sort Product Name, Category, Date Update, and Price order by ASC or DESC

## Installation Steps

1. Install the Golang and GO Environment

```bash
https://golang.org/doc/install
```

2. Clone the repository

```bash
git clone git@github.com:wsaefulloh/rest-api-golang.git (go get)
```

3. Run the app

```bash
go run main.go
```

4. You are all set!

```bash
View the website at: http://localhost:9000
```
